## 🛠️ 技术架构

### 前端技术栈
```
Vue 3 (Composition API) + TypeScript
├── Vite 5 (构建工具)
├── Element Plus 2 (UI组件库)
├── Pinia 2 (状态管理)
├── Vue Router 4 (路由)
└── Axios (HTTP客户端)
```

### 地图服务
- **主要**：高德地图 JavaScript API
- **备用**：天地图
- **功能**：省份标记、旅行路径、飞行动画

### 数据存储
- **开发期**：LocalStorage + 静态 JSON
- **生产期**：Supabase/PostgreSQL + Redis 缓存

### 部署方案
- **静态部署**：Vercel / GitHub Pages
- **容器化**：Docker + Nginx
- **CDN**：静态资源加速
